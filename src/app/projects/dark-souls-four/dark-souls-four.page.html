Projects Dark Souls 4 2D Action Platformer Demo created for the GameBoy Advance
What is Dark Souls IIII? This game was my final project submission for CS 2261 (Media Device Architecture). It was a solo project. This game, which was coded to run on the GameBoy Advance (and has been run on real GameBoy hardware), was written entirely in C. It was compiled using the GCC. Not all art assets were original. Most were taken from various internet resources and manipulated in photoshop. The main character sprite (Solaire) and background art was completely original. I wrote multiple external programs (Mainly in MATLAB, as I was a Teaching Assistant for the language at the time) to assist me in content generation and converting images into compilable C code. This project was meant to provide a rougly 5 minute gameplay experience; development took approximately 1.5 months.
